# NEKO_ERP 客户端 0.1 预览版试跑清单

更新时间: 2026-04-23
适用范围: `erp-client` 预览版试点验证

## 1. 使用方式

本清单用于发布前最后一轮人工试跑。

建议在干净环境中执行，严格按顺序确认，不要跳步。

## 2. 启动前检查

1. 服务端包已经启动
2. 当前客户端连接地址为 `http://127.0.0.1:8080/api`
3. 默认管理员账号 `admin / admin` 可用
4. 当前测试电脑可以正常打开 Electron 客户端

## 3. 必过主路径

### 3.1 登录与首页

1. 打开客户端
2. 登录页输入 `admin / admin`
3. 登录成功后进入首页
4. 首页首跑检查清单可以正常渲染
5. 首页可以看到服务端在线状态

### 3.2 基础主数据

1. 打开公司模块并新建一条记录
2. 打开伙伴模块并新建一条记录
3. 打开产品模板模块并新建一条记录
4. 打开产品变体模块并新建一条记录

### 3.3 主链模块

1. 打开销售订单模块并新建一条记录
2. 打开采购订单模块并新建一条记录
3. 打开账单模块并确认列表可加载

### 3.4 详情页

1. 从列表点开任意主链对象
2. 确认对象进入独立详情页，而不是在原列表中混乱展开
3. 确认详情页可以看到附件区、时间轴区、文档区和动作区
4. 使用返回动作或 `Alt+Left` 返回列表

### 3.5 附件与文件

1. 在详情页打开附件导入
2. 可以正常拉起本机文件选择器
3. 可见合同、发票、通用附件等资料入口

### 3.6 设置与账户

1. 打开设置页
2. 确认连接设置、当前账户、用户管理、角色管理入口存在
3. 打开快捷键设置并完成一次真实键盘录制
4. 切换硬件加速开关后，可看到“需要重启后生效”的提示

### 3.7 关于与手册

1. 打开关于页
2. 确认客户端与服务端手册都能正常阅读
3. 确认关于页路线图与当前 0.1 预览版状态一致

### 3.8 退出

1. 执行退出登录
2. 返回登录页
3. 再次登录仍可正常进入系统

## 4. 阻断发布的问题

出现下面任一问题，应暂停发包:

1. 无法登录
2. 首页首跑检查清单无法渲染
3. 任意核心列表无法加载
4. 详情页无法打开
5. 附件文件选择器无法调用
6. 设置页无法保存关键配置
7. 关于页手册为空白或报错

每一步建议记录:

1. 操作时间
2. 执行人
3. 是否通过
4. 截图或错误描述
5. 是否阻断发布
