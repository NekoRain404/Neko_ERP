# NEKO_ERP 客户端 0.1 预览版发布说明

更新时间: 2026-04-23
适用范围: `erp-client` Electron 桌面客户端
版本: `0.1.0-preview.1`

## 1. 本次发布的定位

这不是正式生产版。

本次交付的目标是:

1. 给小范围试点团队提供可安装、可登录、可试跑的桌面 ERP 客户端
2. 覆盖首批主线:
   - 公司
   - 伙伴
   - 产品模板
   - 产品变体
   - 销售订单
   - 采购订单
   - 账单核查
   - 用户与角色管理
3. 为后续正式版验证首跑路径、桌面体验和回退方式

## 2. 本次发布已经收口的内容

- 首页已经提供 `首跑检查清单`
- 核心记录默认打开独立详情页，不再在列表里拥挤展开
- 详情页已接入附件、时间轴、文档与下一步动作提示
- 设置页已经收口为 `0.1 基础管理启动台`
- 用户管理与角色管理已经具备独立工作台
- 快捷键已支持真实键盘录制
- 硬件加速开关已支持保存和重启提示
- 关于页已内嵌客户端与服务端手册
- 发布包已附带客户端试跑清单，方便试点前最后核查

## 3. 推荐试跑顺序

建议按下面顺序验证客户端:

1. 打开客户端并确认连接地址
2. 使用 `admin / admin` 登录
3. 在首页确认服务端状态和当前会话
4. 新建公司主档
5. 新建伙伴主档
6. 新建产品模板
7. 新建产品变体
8. 新建销售订单
9. 新建采购订单
10. 核查账单对象
11. 检查用户与角色管理

## 4. 交付时必须说明给试点用户的事项

- 先启动服务端包，再启动客户端包
- 默认后端地址为 `http://127.0.0.1:8080/api`
- 当前版本适合小范围试跑，不适合正式生产上线
- 如果硬件加速设置有改动，保存后必须重启客户端
- 如果某个模块被关闭，需要到设置页恢复，不要误判为系统故障

## 5. 已知限制

- 当前仍依赖独立服务端和基础设施，不是完全离线单机版
- 深会计、深制造、深 HR、深报表不在本次预览版主范围
- 当前目标是“可用可试跑”，不是“全模块正式交付”
- 当前更适合试点验证主链，而不是全员铺开

## 6. 试点负责人最少检查项

发布前至少确认下面几点:

1. 登录页可用
2. 首页首跑检查清单顺序清晰
3. 列表可加载
4. 新建与保存可用
5. 详情页可打开
6. 附件可选择文件
7. 快捷键录制可用
8. 设置页连接与账户安全可用
9. 关于页手册可阅读
10. 退出登录可用
